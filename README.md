## 내용

| 대상 | 단원 |
|---|---|
| 초등 1학년 | 수 세기, 덧셈, 뺄셈 |
| 초등 2학년 | 두 자리 수, 덧셈·뺄셈 |
| 초등 3학년 | 곱셈, 나눗셈 기초 |
| 초등 4학년 | 큰 수, 곱셈·나눗셈 |
| 초등 5학년 | 분수, 소수 |
| 초등 6학년 | 비율, 비례 |
| 중학교 1학년 | 정수·유리수 |

## 학습 방식

- 🇫🇮 **핀란드식** 반복 드릴 — 속도·정확도 훈련
- 10문항 게임 형식, 별점 결과, 유물 컬렉션
- 게임화: 유물 획득, 항해 일지, 나침반 힌트
