1 극한값 lim(n→∞) (3n²+2n-1)/(n²-n+2)를 구하시오.
풀이
📌 원리: ∞/∞ 꼴에서는 분자·분모를 최고차항으로 나눈다. 분모의 차수 = 분자의 차수이면 극한 = 최고차항의 계수비.
분자·분모를 n²으로 나누면:
= (3 + 2/n - 1/n²) / (1 - 1/n + 2/n²)
n→∞이면 1/n, 1/n² → 0
= (3+0-0)/(1-0+0) = 3
3
2 극한값 lim(n→∞) (√(n²+n) - n)을 구하시오.
풀이
📌 원리: ∞-∞ 꼴은 유리화(켤레 곱하기)로 분수형으로 변환한다.
√(n²+n) - n = [√(n²+n) - n] × [√(n²+n) + n] / [√(n²+n) + n] = [(n²+n) - n²] / [√(n²+n) + n] = n / [√(n²+n) + n]
분자·분모를 n으로 나누면:
= 1 / [√(1+1/n) + 1]
n→∞: = 1 / (√1 + 1) = 1/2
1/2
3 등비수열 {rⁿ}에서 lim(n→∞) rⁿ의 수렴 조건과 극한값을 구하시오.
풀이
📌 원리: 등비수열의 극한은 공비 r의 범위에 따라 결정된다.
r > 1 : rⁿ → ∞ (발산) r = 1 : rⁿ → 1 (수렴) -1 < r < 1: rⁿ → 0 (수렴) ← 핵심! r = -1 : 진동 (발산) r < -1 : 발산
수렴 조건: -1 < r ≤ 1
r=1이면 극한값 = 1, |r|<1이면 극한값 = 0
-1 < r ≤ 1 일 때 수렴
