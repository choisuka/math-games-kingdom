2010 수능문제 1
limn→∞ (√(9n²+3n+1) − 3n) 의 값은?
① 1/4  ② 1/3  ③ 1/2  ④ 2/3  ⑤ 3/4
📌 √(an²+bn)−√(an²) 유형 → 분자 유리화 후 n으로 나눈다
[풀이]
분자·분모에 켤레식을 곱한다:
(√(9n²+3n+1)−3n)·(√(9n²+3n+1)+3n)/(√(9n²+3n+1)+3n)
= (9n²+3n+1−9n²)/(√(9n²+3n+1)+3n)
= (3n+1)/(√(9n²+3n+1)+3n)

분자·분모를 n으로 나누면:
= (3+1/n)/(√(9+3/n+1/n²)+3)
n→∞: 3/(3+3) = 3/6 = 1/2 ③
2011 수능문제 2
수열 {an}이 a1=2, an+1=√(an+2) 를 만족할 때,
limn→∞ an 의 값을 구하여라. (단, 모든 항은 양수)
📌 점화식 극한: 극한이 L이라 놓으면 L=f(L)로 고정점 방정식을 세운다
[풀이]
극한 L = lim an이 존재한다고 가정하면
양변에 n→∞ 극한: L = √(L+2)
제곱: L²=L+2 → (L-2)(L+1)=0
L>0이므로 L=2

[단조 수렴 확인]
f(x)=√(x+2)로 놓으면, f'(x)=1/(2√(x+2)), x=2에서 f'(2)=1/4 < 1
→ 2 근방에서 수축 사상이므로 an→2로 수렴 ✓
답: 2
2012 9월문제 3
limn→∞ {n·(√(1+1/n) − 1)} 의 값을 구하여라.
📌 limh→0 (f(1+h)−f(1))/h = f'(1) 꼴로 변환
[방법 1 — 유리화]
n·(√(1+1/n)−1) = n·(1/n)/(√(1+1/n)+1) = 1/(√(1+1/n)+1)
n→∞: 1/(1+1) = 1/2

[방법 2 — 미분 정의]
h=1/n으로 치환하면 h→0+
= (√(1+h)−1)/h → (d/dx √x)|_{x=1} = 1/(2√1) = 1/2
답: 1/2
2013 수능문제 4
등비수열 {an}(공비 r, 0<r<1)에 대해
Σn=1∞an=3,   Σn=1∞an²=2 일 때, 첫째항 a1의 값은?
① 4/9  ② 8/13  ③ 12/11  ④ 16/17  ⑤ 4/3
📌 Σan=a₁/(1−r), Σan²=a₁²/(1−r²) (공비 r²인 등비급수)
[풀이]
a₁/(1−r)=3 ……①
a₁²/(1−r²)=2 ……②

②÷①: a₁/(1+r)=2/3 ……③
①에서 a₁=3(1−r), ③에 대입:
3(1−r)/(1+r)=2/3
9(1−r)=2(1+r) → 9−9r=2+2r → 15=17r → r=15/17 (0<1 ✓)
a₁=3(1−15/17)=3·2/17=6/17

검산: a₁/(1−r)=(6/17)/(2/17)=3 ✓
답: 6/17
2014 6월문제 5
limn→∞ (1/n) Σk=1n (k/n) / (1+(k/n)²) 의 값을 구하여라.
📌 리만합: limn→∞(1/n)Σf(k/n) = ∫₀¹f(x)dx
[풀이]
f(x)=x/(1+x²)으로 놓으면
lim = ∫₀¹ x/(1+x²)dx

u=1+x², du=2x dx 치환 (x:0→1, u:1→2):
= (1/2)∫₁² (1/u)du = (1/2)[ln u]₁² = (1/2)·ln2

2014 9월문제 6
양의 실수 전체에서 정의된 함수 f(x)에 대해
limn→∞ n·f(1/n) = 3 일 때,

[풀이]
t=1/n으로 치환하면 n→∞ ↔ t→0+

2015 수능문제 7
수열 {an}이 a1=1이고, 모든 자연수 n에 대해
2a_{n+1} = a_n + a_{n+2} 이면서 limn→∞ an/n = 2 일 때,
a1·a2의 값을 구하여라. (단, {an}은 등차수열)
📌 등차수열 an=a₁+(n-1)d, lim an/n=d

2an+1=an+an+2는 등차수열 조건 (공차 d 일정)

lim an/n = lim [1/n+(1-1/n)d] = d = 2
따라서 a1=1, a2=1+2=3
a₁·a₂=1·3=3
2016 6월문제 8
limn→∞ (√n·(√(n+1)−√n)) 의 값은?
① 0  ② 1/4  ③ 1/2  ④ 1  ⑤ 발산
[풀이]
√n·(√(n+1)−√n) = √n · 1/(√(n+1)+√n)
(유리화: √(n+1)−√n = 1/(√(n+1)+√n))
= √n/(√(n+1)+√n) = 1/(√(1+1/n)+1)
n→∞: 1/(1+1) = 1/2

2016 수능문제 9
limn→∞ (1/n²) Σk=1n k·√(1−(k/n)²) 의 값을 구하여라.
📌 일반화 리만합: (1/n²)Σk·f(k/n) = ∫₀¹x·f(x)dx
[풀이]
k/n=x, 1/n²=dx·(1/n)이고
Σ(k/n)·f(k/n)·(1/n) = ∫₀¹ x·f(x)dx (f(x)=√(1-x²))
= ∫₀¹ x√(1-x²)dx

u=1-x², du=-2x dx (x:0→1, u:1→0):
= (-1/2)∫₁⁰√u du = (1/2)∫₀¹√u du = (1/2)·[2u^(3/2)/3]₀¹ = 1/3
답: 1/3
2017 수능문제 10
x>0, x≠1에서 정의된 함수 f(x)=limn→∞ (xn−x-n)/(xn+x-n) 에 대해,
limx→1+f(x) + limx→1-f(x) 의 값은?
① -2  ② -1  ③ 0  ④ 1  ⑤ 2
[x>1일 때] 분자·분모를 xn으로 나누면:
f(x) = (1−x-2n)/(1+x-2n). x>1이므로 x-2n→0
→ f(x)=1/1=1, 따라서 limx→1+f(x)=1

[x<1일 때] 분자·분모를 x-n으로 나누면:
f(x) = (x2n−1)/(x2n+1). 0<x<1이므로 x2n→0
→ f(x)=-1/1=-1, 따라서 limx→1-f(x)=-1

합: 1+(-1)=0 ③

2019 6월문제 14
수열 {an}이 a1=1, a2=2이고
n≥3일 때 an=2an-1−an-2+2 를 만족한다.
limn→∞ an/n² 의 값을 구하여라.
[풀이]

2020 수능문제 15
양수 a에 대해 수열 {an}이
an = limm→∞ (a·mⁿ+1)/(mⁿ+1) 로 정의될 때,
limn→∞ an 의 값을 구하여라. (단, a≠1)

2020 9월문제 16
limn→∞ (1/n) Σk=1n ln(1+k/n) 의 값을 구하여라.
[풀이]
f(x)=ln(1+x)에 대한 리만합:
lim = ∫₀¹ ln(1+x)dx

부분적분: u=ln(1+x), dv=dx
= [x·ln(1+x)]₀¹ − ∫₀¹ x/(1+x)dx
= ln2 − ∫₀¹ (1−1/(1+x))dx
= ln2 − [x−ln(1+x)]₀¹
= ln2 − (1−ln2) = 2ln2−1

2021 수능문제 17
limn→∞ (1/n) Σk=1n √(1−(k/n)²) 의 값은?
(단, 단위원의 넓이는 π이다)
① π/6  ② π/5  ③ π/4  ④ π/3  ⑤ π/2
📌 ∫₀¹√(1-x²)dx = 반지름 1인 원의 1사분면 넓이 = π/4
[풀이]
f(x)=√(1-x²)에 대한 리만합:

x=sinθ 치환 (x:0→1, θ:0→π/2):

2022 수능문제 18
수열 {an}이 an=(2n+1)/(n²+n) 일 때,
limn→∞ (a1+a2+…+an) 의 값을 구하여라.
[풀이 — 부분분수]
an=(2n+1)/(n(n+1)) = (n+1+n)/(n(n+1)) = 1/n + 1/(n+1)
대기 수열... 실제로: 2n+1=n(n+1)-(n-1)n... 다시 계산:
(2n+1)/(n(n+1)) = A/n + B/(n+1)
A(n+1)+Bn=2n+1 → A+B=2, A=1 → B=1
따라서 an=1/n+1/(n+1)

Sn=Σ(1/k+1/(k+1)) (k=1 to n)
= (1+1/2)+(1/2+1/3)+...+(1/n+1/(n+1))
= 1 + 2·(1/2+1/3+...+1/n) + 1/(n+1)
→ 급수는 발산 (조화급수 포함)

※ 이 급수는 발산합니다. 문제 조건 수정: lim an의 값 구하기
lim an = lim (2n+1)/(n²+n) = lim 2/n = 0
2023 수능문제 19
Σn=1∞an=8,  Σn=1∞an²=4 를 만족하는 양항 등비수열 {an}에 대해
Σn=1∞a2n 의 값을 구하여라.

첫째항 a₁, 공비 r (0<r<1)

a₁²/(1-r²)=4 ……②

②÷①: a₁/(1+r)=1/2 ……③
①: a₁=8(1-r) ……④
③④: 8(1-r)/(1+r)=1/2
16(1-r)=1+r → 15=17r → r=15/17
a₁=8·(2/17)=16/17

Σa2n: 첫째항 a₂=a₁r=(16/17)·(15/17)=240/289, 공비 r²=(15/17)²
= a₂/(1-r²) = (240/289)/((289-225)/289) = 240/64 = 15/4

2025 수능문제 20
수열 {an}이 a1=1이고 모든 n에 대해
an+1 = an/(1+n·an) 을 만족할 때,
limn→∞ n·an 의 값을 구하여라.
📌 역수 치환: bn=1/an으로 놓으면 선형 점화식이 나타날 수 있다
[풀이]
bn=1/an으로 치환:
1/an+1 = (1+n·an)/an = 1/an+n
bn+1=bn+n

b1=1/a1=1
bn=b1+Σk=1n-1k = 1+(n-1)n/2

n·an=n/bn=n/(1+n(n-1)/2)=2n/(2+n(n-1))
= 2n/(n²-n+2) ≈ 2n/n² = 2/n → 0
답: 0

2011 수능문제 22
Σn=1∞ 2/((2n-1)(2n+1)) 의 값을 구하여라.

[풀이]

2012 9월문제 23

[풀이]

따라서 각 항 = 1/(√n·√(n+1)·(√(n+1)+√n))

분자·분모에 (√(n+1)−√n)을 곱하면:
= (√(n+1)−√n)/(√n·√(n+1)·1) = 1/√n − 1/√(n+1)

Σ = (1−1/√2)+(1/√2−1/√3)+... = 1 (망원급수, n→∞)
답: 1

수열 {an}에 대해 Sn=Σk=1nak=3·(1/2)n+c 일 때,

limx→0 (ex-e-x-2sin x) / x³ 의 값을 구하여라.
[방법 — 테일러 전개]
ex=1+x+x²/2+x³/6+...

ex-e-x=2x+2x³/6+...=2x+x³/3+...
sin x=x-x³/6+...
2sin x=2x-x³/3+...

분자=2x+x³/3+...-2x+x³/3+...=2x³/3+...
(ex-e-x-2sinx)/x³ → 2/3
