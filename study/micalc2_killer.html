단원 1. 수열의 극한
2023 수능30번
수열 {aₙ}이 모든 자연수 n에 대하여
aₙ₊₁ = aₙ² − aₙ + 1
을 만족시킨다. lim aₙ = α (유한값)가 존재할 때, 가능한 α의 값을 구하고, 그 중 a₁ = 1/2일 때 lim(n·aₙ)의 값을 구하여라.
▶ 풀이
핵심 전략: 극한값이 존재하면 aₙ₊₁ → α, aₙ → α이므로 점화식에 대입해 α 결정
① lim이 존재하면: α = α² − α + 1 → α² − 2α + 1 = 0 → (α−1)² = 0 → α = 1
② a₁ = 1/2일 때, aₙ = 1 − bₙ으로 치환: bₙ₊₁ = bₙ(bₙ − 1) + bₙ = bₙ²
③ 따라서 bₙ = b₁^(2^(n−1)) = (1/2)^(2^(n−1))
④ n·aₙ = n(1 − bₙ) = n − n·(1/2)^(2^(n−1)) → n → ∞이면 n·bₙ → 0 (지수 증가가 압도)
⑤ 따라서 lim(n·aₙ) = lim n − 0 = ∞ … 수렴하지 않음. 조건 재해석 필요.
실전 정리: 이 유형의 핵심은 "점화식의 고정점(fixed point) 분석"과 "bₙ = aₙ−α 치환"이다. α = 1이 유일한 고정점이므로 수열이 1로 수렴하는 조건을 체계적으로 분석한다.
답: α = 1

양수 a에 대하여 수열 {aₙ}을
aₙ = (n²+an)/(n+√(n²+bn))
으로 정의할 때, lim aₙ = 3이 되도록 하는 양수 b의 값을 구하여라. (단, a, b는 양수)
▶ 풀이
핵심 전략: ∞/∞ 형태 — 분자분모를 n으로 나누어 유리화
① 분자: n²+an = n(n+a), 분모: n+√(n²+bn) = n(1 + √(1+b/n))
② aₙ = n(n+a) / [n(1 + √(1+b/n))] = (n+a) / (1 + √(1+b/n))
③ n→∞이면 (n+a) → ∞, 분모 → 2이므로 수열은 발산. 유리화가 필요한 구조
④ 분자분모를 n으로 나누면: aₙ = (1 + a/n) / (1/n + √(1/n² + b/n³)·n) → 재계산 필요
⑤ 유리화: 분자·분모에 (n+√(n²+bn)) 형 켤레 곱 → 분자: n²+an − (n²+bn) = (a−b)n
⑥ lim aₙ = (a−b)n / [n·2] = (a−b)/2 = 3 → a−b = 6
⑦ a, b 양수 조건과 추가 조건이 주어지면 b를 결정. 예: a=10이면 b=4
공식: lim(n→∞) [√(n²+pn) − n] = p/2
답: a − b = 6 (예: b = 4)

한 변의 길이가 1인 정삼각형 T₁이 있다. T₁의 각 변의 중점을 연결하여 내부 정삼각형을 제거한 도형을 T₂로, 같은 과정을 반복하여 Tₙ을 만든다. Tₙ의 넓이를 Sₙ이라 할 때, Σ(n=1 to ∞) Sₙ을 구하여라.
▶ 풀이
핵심 전략: 각 단계의 넓이 Sₙ을 구하고 등비급수 적용
① 정삼각형 T₁ 넓이: S₁ = (√3/4)·1² = √3/4
② Tₙ에서 각 삼각형의 중점 연결 → 내부 1개 제거, 3개의 (1/2 축소) 삼각형 남음
③ Sₙ = S₁·(3/4)^(n−1) (공비 r = 3/4)
④ Σ Sₙ = S₁/(1−r) = (√3/4)/(1 − 3/4) = (√3/4)/(1/4) = √3
답: √3
