단원 1. 수열의 극한 ★★★★★
창작30번 유형
두 수열 {aₙ}, {bₙ}이 모든 자연수 n에 대하여
(가) n²aₙ + nbₙ = 3n + 2
(나) naₙ + n²bₙ = n² + 5n + 1
을 만족시킬 때, lim(n→∞) (n·aₙ + bₙ)의 값을 구하여라.
▶ 풀이
핵심 전략: 두 연립방정식을 aₙ, bₙ에 대해 풀고 극한 계산
① 크래머 공식. 계수행렬: |n² n / n n²|, 행렬식 Δ = n⁴−n² = n²(n²−1)
② aₙ = [(3n+2)·n² − n·(n²+5n+1)] / n²(n²−1)
분자 = 3n³+2n²−n³−5n²−n = 2n³−3n²−n = n(2n²−3n−1)
aₙ = (2n²−3n−1) / [n(n²−1)] = (2n²−3n−1)/[n(n−1)(n+1)]
③ n·aₙ = (2n²−3n−1)/(n²−1) → (분자 ÷ n², 분모 ÷ n²) → 2 (n→∞)
④ bₙ = [(n²)(n²+5n+1) − n(3n+2)] / n²(n²−1)
분자 = n⁴+5n³+n²−3n²−2n = n⁴+5n³−2n²−2n
bₙ = (n³+5n²−2n−2) / [n(n²−1)]
→ (n→∞) bₙ ≈ n³/n³ = 1
⑤ lim(n·aₙ + bₙ) = 2 + 1 = 3
함정: naₙ → 0처럼 보이지만 실제로는 2로 수렴. 분자 차수 주의!
답: 3
창작29번 유형
자연수 n에 대하여
lim(n→∞) n^(3/2) · (√(n+1) + √(n−1) − 2√n)
의 값을 구하여라.
▶ 풀이
핵심 전략: 테일러 전개 — √(n±1) = √n·√(1±1/n) 를 2차항까지 전개
① √(n+1) = √n·(1+1/n)^(1/2) ≈ √n[1 + 1/(2n) − 1/(8n²) + ···]
② √(n−1) = √n·(1−1/n)^(1/2) ≈ √n[1 − 1/(2n) − 1/(8n²) + ···]
③ √(n+1)+√(n−1) ≈ √n[2 − 1/(4n²)+···] = 2√n − 1/(4n^(3/2)) + ···
④ √(n+1)+√(n−1) − 2√n ≈ −1/(4n^(3/2))
⑤ n^(3/2) · [−1/(4n^(3/2))] = −1/4
일반화: n^(3/2)·(f(n+h)+f(n−h)−2f(n)) → h²·f''(n)·n^(3/2)/2 (중심차분)
답: −1/4
창작30번 유형
수열 {aₙ}이 a₁=2이고 점화식
aₙ₊₁ = (aₙ + 1/aₙ) / 2
를 만족할 때, 다음 물음에 답하여라.
(1) lim aₙ의 값을 구하여라.
(2) bₙ = aₙ − 1이라 할 때, lim(n→∞) 2ⁿ bₙ의 값을 구하여라.
▶ 풀이
핵심: 이 점화식은 √1 = 1을 구하는 Newton 법. 수렴값 L은 L=(L+1/L)/2 → L²=1 → L=1
① L = lim aₙ → L = (L+1/L)/2 → 2L² = L²+1 → L²=1 → L=1 (a₁>0이므로)
② bₙ = aₙ−1 → bₙ₊₁ = aₙ₊₁−1 = (aₙ+1/aₙ)/2 − 1 = (aₙ²+1−2aₙ)/(2aₙ) = (aₙ−1)²/(2aₙ)
③ bₙ₊₁ = bₙ²/(2aₙ). aₙ→1이므로 bₙ₊₁ ≈ bₙ²/2 (초선형 수렴)
④ cₙ = 2bₙ으로 치환: cₙ₊₁ = 2bₙ₊₁ = bₙ²/aₙ = cₙ²/(4aₙ)
⑤ n→∞: aₙ→1이므로 cₙ₊₁ ≈ cₙ²/4. cₙ = dₙ/4이면 dₙ₊₁ = dₙ²/4... 급격히 0 수렴
⑥ 2ⁿbₙ → 0 (bₙ이 2ⁿ보다 훨씬 빠르게 → 0)
결론: Newton법의 2차 수렴은 2ⁿ이 아닌 2^(2ⁿ) 속도 → 2ⁿbₙ → 0
답: (1) 1, (2) 0
창작29번 유형
수열 {aₙ}이 a₁=1이고 모든 자연수 n에 대하여
√aₙ₊₁ = √aₙ + 1/(n(n+1))
을 만족시킬 때, lim(n→∞) aₙ의 값을 구하여라.
또한 Σ(n=1 to ∞) (aₙ₊₁ − aₙ)의 값을 구하여라.
▶ 풀이
핵심: bₙ = √aₙ으로 치환하면 1계 점화식으로 단순화
① bₙ = √aₙ, b₁=1, bₙ₊₁ = bₙ + 1/(n(n+1))
② bₙ = 1 + Σ(k=1 to n−1) 1/(k(k+1)) = 1 + Σ(k=1 to n−1)(1/k − 1/(k+1)) = 1 + 1 − 1/n = 2 − 1/n
③ lim bₙ = 2 → lim aₙ = (lim bₙ)² = 4
④ Σ(aₙ₊₁−aₙ): 망원급수 → (lim aₙ) − a₁ = 4 − 1 = 3
핵심 기술: √ 치환 후 부분분수 망원급수. 두 물음을 연결해 풀기.
답: lim aₙ = 4, 급수 = 3
창작30번 유형
양의 실수 t에 대하여 수열 {aₙ(t)}를
a₁(t) = t, aₙ₊₁(t) = t · aₙ(t) / (1 + aₙ(t))
로 정의한다. lim(n→∞) n · aₙ(t) = 1이 되도록 하는 t의 값을 구하여라.
▶ 풀이
핵심: bₙ = 1/aₙ 역수 치환으로 선형 점화식 변환
① bₙ = 1/aₙ, b₁ = 1/t
② bₙ₊₁ = (1+aₙ)/(t·aₙ) = 1/(t·aₙ) + 1/t = bₙ/t + 1/t
③ bₙ₊₁ = (1/t)bₙ + 1/t → 선형 점화식 (공비 1/t)
④ t≠1이면: bₙ = (b₁−p)·(1/t)^(n−1)+p (단 p는 고정점: p=1/(t−1))
⑤ t=1이면: bₙ₊₁ = bₙ+1 → bₙ = b₁+(n−1) = 1/t+(n−1) = n (∵ t=1)
⑥ t=1이면 aₙ = 1/bₙ = 1/n → n·aₙ = 1 → lim n·aₙ = 1 ✓
⑦ t≠1이면 aₙ = 1/bₙ → 0 or ∞, n·aₙ → 0 (|t|≠1시 bₙ 기하급수 성장)
결론: 오직 t=1일 때만 lim n·aₙ = 1
답: t = 1
단원 2. 무한급수 ★★★★★
창작30번 유형
반지름이 1인 원에 내접하는 정삼각형 T₁이 있다. T₁의 각 변의 중점을 이어 내부 삼각형을 만들고, 이 내부 삼각형에 내접하는 원을 그린다. 이 원에 내접하는 정삼각형을 T₂라 하고, 같은 과정을 반복한다.
Tₙ의 넓이를 Sₙ, 매 단계에서 그린 원의 넓이를 Cₙ이라 할 때,
Σ(n=1 to ∞)(Sₙ + Cₙ)을 구하여라.
▶ 풀이
핵심: 각 단계 축소 비율 계산 (정삼각형→내접원→내접정삼각형)
① 반지름 R인 원에 내접 정삼각형 한 변: a = R√3
② R=1이면 한 변 a₁ = √3, 넓이 S₁ = (√3/4)(√3)² = 3√3/4
③ 변의 중점 삼각형 변 = a/2 = √3/2 (중점연결정리), 이 삼각형 내접원 반지름 = (변)/(2√3) = (√3/2)/(2√3) = 1/4
④ 정삼각형에서 내접원 반지름 r = a/(2√3). a=√3이면 r=√3/(2√3)=1/2 (T₁의 내접원 = 원래 원과 다름)
⑤ 중점 삼각형의 내접원 반지름 = (a/2)/(2√3) = 1/(4) (a=√3 대입) → R₂ = 1/4
⑥ 비율 공비: R₂/R₁ = (1/4)/1 = 1/4 → 넓이 비 (1/4)² = 1/16
⑦ Cₙ = π·Rₙ² = π·(1/4)^(2(n−1)) = π·(1/16)^(n−1)
⑧ Sₙ = (3√3/4)·(1/16)^(n−1)
⑨ Σ(Sₙ+Cₙ) = (3√3/4+π)·1/(1−1/16) = (3√3/4+π)·16/15
검산: S₁+C₁ = 3√3/4 + π (n=1 항), 공비 1/16
답: 16(3√3/4+π)/15 = (12√3+16π)/15
창작29번 유형
수열 {aₙ}에 대하여 급수 Σ(n=1 to ∞) aₙ이 수렴하고 그 합이 S이다.
다음 조건이 성립할 때, S의 값을 구하여라.
(가) Σ(n=1 to ∞) n·aₙ = 3S − 1
(나) aₙ = (2/3)^(n−1) · (αn + β) (α, β는 상수)
(다) a₁ = 1
▶ 풀이
핵심: aₙ의 형태를 이용해 S, Σn·aₙ을 α, β로 표현 후 연립
① a₁ = α+β = 1 ...(1)
② S = Σ(2/3)^(n−1)(αn+β) = α·Σn(2/3)^(n−1) + β·Σ(2/3)^(n−1)
③ Σ(2/3)^(n−1) = 1/(1−2/3) = 3
④ Σn·x^(n−1) = 1/(1−x)² at x=2/3: 1/(1/3)² = 9 → Σn(2/3)^(n−1) = 9
⑤ S = 9α + 3β ...(2)
⑥ Σn·aₙ = Σn(2/3)^(n−1)(αn+β) = αΣn²(2/3)^(n−1) + βΣn(2/3)^(n−1)
⑦ Σn²x^(n−1) = (1+x)/(1−x)³ at x=2/3: (5/3)/(1/3)³ = (5/3)·27 = 45
⑧ Σn·aₙ = 45α + 9β ...(3)
⑨ 조건 (가): 45α+9β = 3(9α+3β)−1 = 27α+9β−1 → 18α = −1 → α = −1/18
⑩ (1)에서 β = 1−α = 1+1/18 = 19/18
⑪ S = 9(−1/18)+3(19/18) = −1/2+19/6 = −3/6+19/6 = 16/6 = 8/3
답: S = 8/3
창작30번 유형
급수 Σ(n=1 to ∞) 1/[n(n+1)(n+2)]의 값을 구하여라.
또한 Σ(n=1 to ∞) n/[(n+1)!]의 값도 구하여라.
▶ 풀이
핵심: 이중 부분분수 분해 후 망원급수, 두 번째는 팩토리얼 망원
[첫 번째 급수]
① 1/[n(n+1)(n+2)] = (1/2)[1/(n(n+1)) − 1/((n+1)(n+2))]
② Sₙ = (1/2)[1/(1·2) − 1/((n+1)(n+2))] → S = (1/2)·(1/2) = 1/4
[두 번째 급수]
③ n/[(n+1)!] = [(n+1)−1]/[(n+1)!] = 1/n! − 1/(n+1)!
④ 망원급수: Σ(1/n! − 1/(n+1)!) = 1/1! = 1
변형: Σn/(n+1)! = e−2 (다른 방법으로 계산 가능. 위 방법이 더 직관적)
답: 1/4 · · · 1
창작28번 유형
양의 정수 m에 대하여 급수
f(m) = Σ(n=1 to ∞) 1/(n(n+m))
가 수렴할 때, Σ(m=1 to ∞) f(m)/m의 값을 구하여라.
▶ 풀이
핵심: 부분분수 → f(m) 닫힌 형식 → 다시 급수 계산
① 1/(n(n+m)) = (1/m)[1/n − 1/(n+m)]
② f(m) = (1/m)·Σ(n=1 to ∞)[1/n−1/(n+m)] = (1/m)·(1+1/2+···+1/m) = H_m/m
(단 H_m = 1+1/2+···+1/m = m번째 조화수)
③ f(m)/m = H_m/m² → Σ(m=1 to ∞) H_m/m²
④ 이는 알려진 급수: Σ H_m/m² = 2ζ(3) (Apéry 상수의 항등식)
⑤ 수능/교육청에선 이 무한급수를 닫힌 형식으로 요구하지 않으므로, 유한 부분합으로 의미 파악이 목적
핵심 기술: 이중 급수에서 f(m)의 닫힌 형식 도출 → 조화수 개념
답: f(m) = H_m/m (여기서 H_m은 m번째 조화수)
창작30번 유형
등비수열이 아닌 수열 {aₙ}이 다음 조건을 모두 만족할 때, Σ(n=1 to ∞) aₙ을 구하여라.
(가) a₁ = 1, a₂ = 1/2
(나) n≥3일 때 aₙ = aₙ₋₁/2 + aₙ₋₂/4
(다) Σaₙ이 수렴한다.
▶ 풀이
핵심: 특성방정식으로 일반항 → 수렴 조건 → 합
① 특성방정식: r² = r/2 + 1/4 → r² − r/2 − 1/4 = 0 → 4r²−2r−1=0
② r = (2±√(4+16))/8 = (2±√20)/8 = (1±√5)/4
③ r₁=(1+√5)/4 ≈ 0.809, r₂=(1−√5)/4 ≈ −0.309 (둘 다 |r|<1)
④ aₙ = A·r₁^(n−1) + B·r₂^(n−1)
⑤ a₁=1: A+B=1, a₂=1/2: A·r₁+B·r₂=1/2
⑥ 연립 풀면 A=(1/2−r₂)/(r₁−r₂), B=(r₁−1/2)/(r₁−r₂)
⑦ Σaₙ = A/(1−r₁) + B/(1−r₂)
⑧ Σaₙ에 Σ(가)를 직접 사용: S = a₁+a₂+S/2+S/4 → S = 1+1/2+3S/4 → S/4=3/2 → S=6
단계 ⑧ 핵심 아이디어: n≥3항들의 합을 점화식으로 나타내면 S−a₁−a₂ = (S−a₁)/2 + S/4
답: 6
단원 3. 미분법 ★★★★★
창작29번 유형
함수 f(x) = e^x·cos(x)에 대하여 f⁽ⁿ⁾(0)의 일반 공식을 구하고,
Σ(n=0 to ∞) f⁽ⁿ⁾(0)/n! · xⁿ 이 수렴하는 범위와 합을 구하여라.
▶ 풀이
핵심: eˣcos(x)의 n차 도함수 공식: (√2)ⁿ·eˣ·cos(x+nπ/4)
① f⁽ⁿ⁾(x) = (√2)ⁿ·eˣ·cos(x + nπ/4) (복소수 지수 이용 유도)
② f⁽ⁿ⁾(0) = (√2)ⁿ·cos(nπ/4)
③ 테일러 급수: Σ f⁽ⁿ⁾(0)/n! · xⁿ = Σ (√2)ⁿcos(nπ/4)xⁿ/n!
④ 이는 Re[Σ(√2·e^(iπ/4)·x)ⁿ/n!] = Re[e^(√2·e^(iπ/4)·x)]
⑤ √2·e^(iπ/4) = √2·(1/√2 + i/√2) = 1+i
⑥ e^((1+i)x) = eˣ·e^(ix) = eˣ(cosx + i·sinx)
⑦ Re 부분 = eˣcosx = f(x) → 수렴 범위: 모든 실수, 합 = eˣcosx
답: f⁽ⁿ⁾(0) = (√2)ⁿcos(nπ/4), 급수합 = eˣcosx (수렴: 전 실수)
창작30번 유형
함수 f(x) = ln|x² − x − 2|에 대하여
(가) f(x)의 미분불가능점을 모두 구하여라.
(나) f(x)의 극값을 구하여라.
(다) f'(x) = 0의 해를 모두 구하여라.
▶ 풀이
핵심: x²−x−2 = (x−2)(x+1). 이것의 부호 분석이 핵심
① x²−x−2 = (x−2)(x+1) = 0 → x=2 or x=−1 (정의역 제외, 미분불가능점)
② f'(x) = (2x−1)/(x²−x−2) (x≠−1, 2)
③ f'(x)=0 → 2x−1=0 → x=1/2
④ 구간 분석: x<−1에서 x²−x−2>0 → f정의됨
x=1/2에서 f'(0→) 부호 확인: 1/2<2이므로 −1
x²−x−2 = (x−2)(x+1) < 0 이므로 f(x) = ln|negative| = ln(x²−x−2의 절댓값)
⑤ x=1/2: 분자 2(1/2)−1=0, 분모 (1/4−1/2−2)<0 → f'(1/2)=0 ✓
⑥ x=1/2 근방: f'(x)=(2x−1)/(negative) → x<1/2일 때 f'>0, x>1/2일 때 f'<0 → 극댓값
⑦ f(1/2) = ln|1/4−1/2−2| = ln(9/4) = 2ln(3/2)
주의: −10 항상
답: (가) x=−1, x=2 / (나) 극댓값 2ln(3/2) at x=1/2 / (다) x=1/2
창작30번 유형
함수 f(x)가 f(f(x)) = x (f의 역함수 = f 자신) 를 만족하고 f가 미분가능할 때,
(가) f(3) = 5, f(5) = 3 이라 할 때 f'(3)·f'(5)의 값을 구하여라.
(나) f(a) = a인 고정점 a에서 f'(a)의 가능한 값을 모두 구하여라.
▶ 풀이
핵심: f(f(x))=x의 양변 미분 → 연쇄법칙
① f(f(x))=x의 양변을 x로 미분: f'(f(x))·f'(x) = 1
② (가): x=3 대입: f'(f(3))·f'(3) = 1 → f'(5)·f'(3) = 1
③ 따라서 f'(3)·f'(5) = 1
